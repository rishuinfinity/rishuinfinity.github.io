Emotion Monitoring Models
  • Offline video Mode
    • This mode is applicable when you have the video downloaded on your laptop/mobile and you want to run this emotion monitoring for that video.
    • When the page is loaded, then on pressing start, video starts playing and cant be stopped in the middle. If you want to restart then refresh the page.
    • When the video is completed then you will be prompted to download a csv file with all the recorded emotion data.
    • The csv file can be used to extract any kind of further intutions from the data.
  • Online video Mode
    • Under construction
  • No-video Mode
    • This mode can be used just to get an intution/feel of the model. It dynamically plots the received emotion values.
